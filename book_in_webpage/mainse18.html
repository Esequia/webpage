4.4.1 Algoritmo para fatoração LU

O algoritmo para fatoração pode ser escrito como

 1. function [L,A]=fatoraLU(A)  
 2.     n=size(A,1)  
 3.     L=eye(n,n)  
 4.     for j=1:n-1  
 5.         for i=j+1:n  
 6.             L(i,j    )=A(i,j)/A(j,j)  
 7.             A(i,j+1:n)=A(i,j+1:n)-L(i,j)*A(j,j+1:n)  
 8.             A(i,j    )=0  
 9.         end  
10.     end  
11. endfunction
