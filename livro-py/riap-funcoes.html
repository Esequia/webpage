A.5 Funções

Além das muitas funções disponíveis em Python (e os tantos muitos pacotes livers disponíveis), podemos definir nossas próprias funções. Para tanto, existe a instrução def. Veja os seguintes exemplos:

Exemplo A.5.1. O seguinte código:

def f(x):  
    return x + np.sin(x)
